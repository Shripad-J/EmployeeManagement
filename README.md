# Employee Management System

This project is a Java-based Employee Management System using Hibernate for database operations.

## Features
- Add new employee
- View employee list
- Update employee details
- Delete employee records

## Technologies Used
- Java
- Hibernate
- MySQL
- Maven
- Eclipse

## How to Run
1. Clone the repository
2. Import the project into Eclipse
3. Configure database in hibernate.cfg.xml
4. Run the application
